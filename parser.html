Parser

Tree-Sitter를 활용한 소스코드 구조 분석 및 파싱 시스템

개발 순서 1단계

개요

Tree-Sitter는 소스 코드를 AST (추상 구문 트리)로 변환하는 고성능 파싱 라이브러리입니다. RAG 시스템에서 코드의 구조적 정보를 정확하게 추출하기 위해 사용됩니다.

단순히 텍스트로 코드를 처리하는 것이 아니라, 클래스, 메서드, 변수 등의 문법 구조를 이해하고 분석할 수 있습니다.

구현 상세

구분 내용 상세 설명
핵심 엔진 Tree-Sitter 소스 코드를 AST (추상 구문 트리)로 변환하는 고성능 파싱 라이브러리 설치
구현 언어 Python RAG 파이프라인(FastAPI, LlamaIndex)이 Python 기반이므로, 공식 Python 바인딩인 tree-sitter 라이브러리를 사용
언어 문법
tree-sitter-java tree-sitter-sql tree-sitter-html tree-sitter-css tree-sitter-javascript tree-sitter-dart tree-sitter-yaml
분석 대상 언어 (Java, SQL, HTML, CSS, JavaScript, JSP, Dart, YAML 등)의 공식 문법 정의를 로드
실행 환경 Linux 또는 WSL Tree-Sitter 설치 및 빌드 과정에서 C 컴파일러가 필요할 수 있으므로 Linux 환경이 가장 안정적

설치 방법

1. Python Tree-Sitter 라이브러리 설치

pip install tree-sitter

2. 언어별 문법 파서 설치

# Java
pip install tree-sitter-java

# SQL
pip install tree-sitter-sql

# JavaScript
pip install tree-sitter-javascript

# HTML/CSS
pip install tree-sitter-html tree-sitter-css

# Dart
pip install tree-sitter-dart

3. 기본 사용 예제

from tree_sitter import Language, Parser
import tree_sitter_java as tsjava

# Java 파서 초기화
JAVA_LANGUAGE = Language(tsjava.language())
parser = Parser(JAVA_LANGUAGE)

# 코드 파싱
source_code = b"""
public class Example {
    public void hello() {
        System.out.println("Hello World");
    }
}
"""

tree = parser.parse(source_code)
root_node = tree.root_node

# AST 탐색
print(root_node.sexp())

RAG 시스템에서의 장점

🎯 정확한 코드 이해

클래스, 메서드, 변수 등 코드의 구조적 요소를 정확하게 식별하여 더 나은 검색 결과를 제공합니다.

⚡ 고성능 파싱

대용량 코드베이스도 빠르게 분석하여 실시간 응답이 가능합니다.

🔍 의미론적 검색

코드의 의미와 맥락을 이해하여 관련성 높은 코드 조각을 찾아냅니다.

🌐 다양한 언어 지원

Java, JavaScript, Python, SQL 등 50개 이상의 프로그래밍 언어를 지원합니다.
