섹션 2 · Ollama 설치 및 설정

Windows 10에서 Ollama 설치 및 설정

  • 설치 전 필수 확인 (사전 준비)

    Windows 10 버전 확인(권장: 22H2 이상), 가상화(VT-x/AMD‑V) 활성화, 모델 저장을 위한 충분한 디스크 여유를 먼저 검증하세요. Ollama 설치파일은 공식 사이트에서 Windows용 'OllamaSetup.exe'를 다운로드합니다(공식 다운로드 페이지에서 최신 릴리스 확인).

  • 설치 방법 — 네이티브 설치와 WSL2(권장)

    네이티브: 다운로드한 'OllamaSetup.exe'을 더블클릭해 설치 마법사를 따라가면 간단히 설치 완료됩니다. WSL2 권장 흐름: 관리자 권한 PowerShell에서 'wsl --install'로 WSL2를 활성화하고 기본 배포판으로 Ubuntu 선택 후, Ubuntu 세션에서 Ollama의 Linux 설치 가이드를 따라 패키지를 설치하세요. WSL2는 Linux 라이브러리 호환성과 GPU 드라이버 연결(특히 CUDA) 면에서 안정적입니다.

  • 설치 후 설정 및 동작 확인

    설치 완료 뒤 시작메뉴 또는 커맨드라인에서 'ollama' 명령이 동작하는지 확인하고, 로컬 웹 UI/서버는 'open-webui serve'로 기동해 서비스 정상 여부를 점검하세요. 모델 다운로드나 서비스 실행 시 디스크 사용량과 GPU 드라이버(CUDA) 호환성을 확인해야 하며, 네이티브 설치는 간단하지만 GPU 가속 사용 시 드라이버 문제 가능성이 있습니다. 요약: WSL2 권장은 호환성·성능 측면에서 우수, 네이티브는 빠른 설치와 간편 운영에 적합합니다.

핵심: 설치 전 환경 체크 → WSL2 권장 → 'open-webui serve'로 동작 확인
