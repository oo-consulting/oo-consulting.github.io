Llama 3.3 & Qwen3 Coder

LLM 듀얼 스택 통합과 응답 품질 최적화 전략

개발 순서 4단계

개요

본 단계는 Llama 3.3 (범용 추론)과 Qwen3 Coder (코드 전문)를 듀얼 스택으로 구성해 RAG 시스템의 응답 품질을 최적화합니다. 질의 유형에 따라 모델을 선택하거나 순차적으로 호출해 답변을 강화합니다.

프롬프트 템플릿 표준화, 컨텍스트 윈도우 관리, 토큰 비용 모니터링, 모델 헬스 체크를 자동화하여 서비스 운영 환경에서도 안정적인 응답 경험을 제공합니다.

프롬프트 표준화 컨텍스트 선택 모델 라우팅 품질 모니터링

구현 상세

구분 내용 상세 설명
모델 라우팅 질의 타입 기반 라우터 자연어 설명, 정책 문서 등은 Llama 3.3으로, 코드 생성/분석은 Qwen3 Coder로 전달하는 규칙 기반 라우터를 구성합니다.
컨텍스트 관리 토큰 창 자동 조절 질문 길이와 검색 결과 토큰을 합산해 모델 최대 토큰을 초과하지 않도록 컨텍스트를 슬라이싱합니다.
프롬프트 체계 system/user 템플릿 분리 System 프롬프트는 답변 톤, 포맷, 금지어를 정의하고 User 프롬프트는 질의와 컨텍스트만 포함하도록 구성합니다.
품질 모니터링 LLM 평가 지표 BLEU, Rouge-L, Code BLEU 등 지표를 배치 평가로 수집하고, 사용자 피드백과 함께 대시보드에 시각화합니다.

설치 및 개발 방법

1. 모델 러너 환경 준비

# Ollama 또는 vLLM 환경 준비 (예: Ollama)
curl https://ollama.ai/install.sh | sh
ollama pull llama3.3
ollama pull qwen3-coder

# Python 클라이언트 패키지 설치
pip install openai httpx pydantic

사내 GPU 클러스터에서는 vLLM 또는 Hugging Face Text Generation Inference를 사용할 수 있습니다.

2. 질의 라우팅 서비스 구현

python services/llm_router.py \
  --llama-endpoint http://localhost:11434/v1 \
  --qwen-endpoint http://localhost:5010/v1 \
  --rule-config ./configs/router_rules.yaml

규칙 파일에는 키워드, 태그, confidence threshold 등을 정의하여 라우팅 정확도를 높입니다.

3. 프롬프트 템플릿 테스트

python tests/prompt_sanity.py \
  --template ./templates/system_prompt.md \
  --queries ./datasets/sample_queries.json \
  --llm llama

샘플 질의를 통해 응답 형식, 토큰 길이, 금지어 위반 여부를 점검합니다. 코드 응답은 linter/formatter로 후처리합니다.

응답 품질 향상을 위한 베스트 프랙티스

컨텍스트 리라이트

  • 유사한 문단이 다수일 경우 상위 3개 문단만 사용
  • 코드 블록은 길이 제한에 맞춰 함수 시그니처 위주로 압축
  • 사용자 질의를 KorBERT 임베딩으로 변환 후 가장 근접한 주석 추가

Guardrail 설계

  • 규칙 기반 필터로 민감 정보(키, 비밀번호) 노출 방지
  • 코드 실행 예제는 반드시 read-only 모드로 제한
  • 금지어/금지 패턴 검출 시 재프롬프트 또는 빈 응답 반환

토큰 비용 최적화

  • 컨텍스트를 Top-K-Weighted 방식으로 정렬해 불필요한 토큰 제거
  • 답변 길이 제한을 QA 유형별로 설정 (예: 코드 답변 300 tokens)
  • 비즈니스 시간대별로 모델/파라미터를 자동 조정

실패 시나리오 대응

  • 모델 응답 지연 시 백업 모델(예: GPT-4o-mini) 호출
  • 오류 메시지는 사용자 친화적 문장으로 가공
  • 빈 답변 발생 시 ChromaDB 재쿼리 후 재시도

RAG 시스템에서의 장점

유연한 답변 전략

질의 특성에 따라 모델을 선택해 설명형, 코드형 질문 모두에 최적 응답을 제공합니다.

정확도 향상

프롬프트 표준화와 컨텍스트 리라이트로 환각을 줄이고 상위 10% 정확도를 확보합니다.

개발자 친화성

코드 생성/수정 요청 시 Qwen3 Coder가 테스트 코드까지 제안해 개발 효율을 높입니다.

운영 신뢰성

Guardrail과 백업 모델 전략으로 오류 상황에서도 안정적인 사용자 경험을 제공합니다.
