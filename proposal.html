System Proposal

RAG 시스템
도입 제안서

RAG(Retrieval-Augmented Generation) 시스템으로 사내 문서와 외부 지식을 통합하여 업무 효율성을 극대화하는 AI 기반 지능형 검색 시스템의 도입을 제안드립니다.

제안서 확인하기 아키텍처 보기
Project Overview

프로젝트 개요

FastAPI 기반 RAG 백엔드와 Tomcat UI를 하나의 Docker Compose 스택으로 통합한 엔터프라이즈 솔루션

🎯

핵심 목표

문서 임베딩 + LLM 추론을 결합해 도메인 지식 기반의 정확한 답변을 제공합니다.

📚

통합 지식베이스

사내 문서와 외부 지식을 통합하고 관리자 UI에서 인제스트, 모니터링, 질의를 간편하게 수행합니다.

🔄

양방향 인제스트

UI와 CLI 모두에서 지식 업데이트가 가능하여 최신화 주기를 단축합니다.

🔒

보안 환경 대응

온라인/오프라인 임베딩 모드, 로컬 LLM(Ollama)으로 네트워크 제한 환경(인터넷이 차단된 사내망)에서도 운영 가능합니다.

✅ 주요 기능
📖 문서 기반 한국어 질의응답 📦 TOPIK, MDN, 한국어 사전, 업무 문서 지원 📄 PDF, DOCX, HWPX, 텍스트, 코드 처리 🔌 오프라인 LLM 및 임베딩 모드 📊 실시간 시스템 모니터링
Expected Benefits

기대 효과

RAG 시스템 도입을 통해 얻을 수 있는 주요 이점

⚡

지식 최신화 주기 단축

UI/CLI 양방향 인제스트로 새로운 문서와 지식을 빠르게 시스템에 반영할 수 있습니다.

🛡️

운영 안정성 확보

Redis 기반 작업 상태 관리와 헬스체크로 시스템의 안정적인 운영을 보장합니다.

🔐

네트워크 제한 환경 대응

오프라인 임베딩 모드와 로컬 LLM으로 보안이 중요한 환경에서도 사용 가능합니다.

📦

통합 지식베이스 구축

다양한 문서 형식(PDF, DOCX, HWPX, 코드)을 지원하여 모든 지식을 통합 관리합니다.

Deployment Guide

배포 및 운용

Docker Compose를 활용한 간편한 배포 및 운영 방법

환경 설정

.env 파일을 생성하고 필요한 환경 변수를 설정합니다. OLLAMA_MODEL, EMBED_MODEL 등의 핵심 설정을 구성합니다.

이미지 빌드

docker compose build

서비스 시작

# 포그라운드 실행
docker compose up

# 백그라운드 실행
docker compose up -d

접속 확인

# UI 대시보드
http://localhost:8080/

# API 상태 확인
curl http://localhost:8000/health
curl http://localhost:8000/rag/status

🎮 GPU 환경 설정

💻

요구사항

• CUDA 12.x 이상
• NVIDIA Container Toolkit 설치
• docker-compose.yml의 deploy.resources 섹션으로 GPU 전달
• GPU 미사용 시 해당 섹션 제거 또는 주석 처리

Configuration

환경 변수 설정

시스템 동작을 제어하는 주요 환경 변수

LLM 및 임베딩 설정

변수 설명 기본값
OLLAMA_HOST Ollama 서버 주소 http://ollama:11434
OLLAMA_MODEL 사용할 LLM 모델 qwen2.5:3b-instruct-q4_K_M
OLLAMA_NUM_CTX LLM 컨텍스트 크기 4096
OLLAMA_REQUEST_TIMEOUT 요청 타임아웃(초) 240
EMBED_MODEL SentenceTransformer 모델 intfloat/multilingual-e5-base
EMBED_MODEL_ALLOW_DOWNLOADS 모델 다운로드 허용 0

검색 설정

변수 설명 기본값
TOP_K 최종 반환 문서 수 5
MIN_RELEVANCE_SCORE 최소 유사도 스코어 0.75
RERANKER_MODEL 재순위화 모델 cross-encoder/ms-marco-MiniLM-L-6-v2
RERANKER_TOP_K 재순위화 전 후보 수 20

인제스트 설정

변수 설명 기본값
INGEST_ALLOW_DOWNLOADS 원격 덤프 다운로드 허용 0
INGEST_BATCH_SIZE Chroma 업서트 배치 크기 4
TOPIK_VOCAB_ENABLED TOPIK 어휘 파이프라인 활성화 1
MDN_ENABLED MDN 파이프라인 활성화 1
KOREAN_DICT_ENABLED 한국어 사전 파이프라인 활성화 1
Extensibility

확장 포인트

새로운 소스 추가 및 시스템 확장 방법

모듈 작성

app/ 디렉토리 아래에 {source}_ingest.py 모듈을 작성합니다.

함수 구현

ensure_{source}_corpus() 함수를 구현하여 데이터 수집 및 전처리 로직을 작성합니다.

백엔드 등록

app/ingest.py에 새로운 소스 선택지를 추가합니다.

UI 반영

frontend/app.js의 소스 목록에 새로운 소스를 추가하면 UI에 자동으로 반영됩니다.

Quick Start

빠른 PoC 실행

1️⃣

.env 파일 설정

환경 변수 파일을 생성하고 기본 설정을 구성합니다.

2️⃣

Docker Compose 실행

UI 접속
