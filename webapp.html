RAG 사용자 인터페이스 구축

FastAPI 게이트웨이를 활용하여 스트리밍 응답, 근거 문서, 피드백 수집을 지원하는 웹 애플리케이션을 구현합니다.

현재 진행률

50% · UI/UX 정립 중

스트리밍 렌더링 · 접근성 · 다국어 지원

개요

Web App은 사용자가 RAG 질의를 수행하고, 스트리밍 응답과 근거 문서를 확인하며, 피드백을 남길 수 있는 상호작용형 인터페이스입니다. Next.js + Tailwind 조합을 사용하여 일관된 디자인과 반응형 레이아웃을 손쉽게 구현합니다.

React Query를 활용해 API 캐싱을 관리하고, Zustand로 UI 상태를 분리하여 유지보수성을 높입니다.

구현 상세

항목 내용 비고
프레임워크 Next.js + Tailwind SSR + CSR 혼합, 디자인 시스템 공유

설치 및 개발 방법

1. 프로젝트 초기화

npx create-next-app@latest rag-webapp --typescript --tailwind --eslint
cd rag-webapp
npm install @tanstack/react-query zustand react-markdown rehype-highlight highlight.js

2. API 연동 훅

export const useRagQuery = () => {
  const client = useQueryClient();
  return useMutation({
    mutationFn: async (payload: RagRequest) => {
      const res = await fetch('/api/rag/query', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(payload),
      });
      if (!res.ok) throw new Error('Query failed');
      return res.json();
    },
    onSuccess: data => client.setQueryData(['history'], prev => [...(prev ?? []), data]),
  });
};

3. 배포

# Vercel
npm run build
vercel deploy --prod

# 사내망 Docker 배포
docker build -t registry.example.com/rag-webapp:latest .
docker push registry.example.com/rag-webapp:latest
kubectl rollout restart deployment rag-webapp
