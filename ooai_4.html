섹션 3 — Llama 3.1 설치 및 설정

Windows 10에서 Llama 3.1 설치 및 초기 설정

  • 설치 전 준비 및 필수 파일

    사전 준비: 관리자 권한의 PowerShell/CMD 실행(또는 WSL2 선택 설치). 필수 설치 파일 예시: - Ollama 설치 프로그램(예: ollama-installer.exe) - 운영체제 기본 터미널(PowerShell/CMD, 관리자 권한) - (GPU 사용 시) NVIDIA 드라이버 설치 파일 및 CUDA Toolkit 설치 파일(예: cuda-xxx-installer.exe) - 충분한 디스크 여유(모델 크기별 상이)와 최소 16GB RAM 권장 모델 파일 자체는 Ollama가 CLI로 자동 내려받습니다(예: llama3.1 8B는 약 4~5GB).

  • Ollama 설치 및 모델 다운로드 절차

    1) Ollama 설치 프로그램(ollama-installer.exe)을 관리자 권한으로 실행하여 설치. 2) 설치 완료 후 터미널에서 모델을 실행·다운로드: ollama run llama3.1 8B — 이 명령은 8B 모델을 내려받고 바로 실행합니다(약 4~5GB). 3) 설치된 모델 확인: ollama list 로 정상 설치 여부와 모델 목록을 검토. 4) 다운로드 실패 시: 네트워크 상태 확인, 디스크 여유 확보 후 재시도하거나 WSL 환경에서 재시도하여 안정성 향상.

  • 운영 권장사항 및 문제 해결 포인트

    - 모델 크기 선택: PoC는 8B 권장(디스크·메모리 부담 적음). 대형 모델(예: 70B)은 더 높은 RAM/GPU 요구. - GPU 활용 시: NVIDIA 드라이버와 CUDA 버전 호환성 사전 점검(불일치는 성능·안정성 문제를 야기). - 권한·환경: 관리자 권한으로 설치·실행, 안정적 네트워크(대용량 다운로드 대비), 충분한 디스크 여유 확보. - 점검 명령: ollama list(설치 확인), 로그 및 오류 메시지 확인 후 재시도 또는 WSL로 전환하여 다운로드 안정화.
