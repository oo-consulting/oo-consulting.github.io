Windows 10에서 Ollama 및 Llama 3.1을 활용한 내부 임베딩 구축 방법

발표 구성: Ollama 설치·설정 → Llama 3.1 설치·운영 → 내부 임베딩 파이프라인 구성 → 성능 최적화 전략 → 운영 중 문제해결 및 실제 적용사례 분석. 전제: Windows 10(WSL2 권장) 기반 로컬 운영으로 데이터 프라이버시 확보. 설치 전 점검 항목: Windows 버전 확인, 가상화(WSL2/Hyper-V) 활성화, GPU/CPU 및 디스크 여유 확인. 구현 포인트: Ollama로 모델 관리·실행, Llama 3.1로 강력한 로컬 추론, 임베딩 생성 → 벡터DB 연동 → 고성능 검색·RAG 파이프라인 구성. 발표자: [조직명 / 담당자명 — 필요에 맞게 기입]. 핵심 요약: 로컬 환경에서 프라이버시를 유지하면서 단계별로 안정적이고 성능 좋은 임베딩 기반 검색 시스템을 구축하는 방법을 제시합니다.
