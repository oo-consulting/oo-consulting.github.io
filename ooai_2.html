목차 — 섹션 1

Windows 10에서 Ollama 및 Llama 3.1을 활용한 내부 임베딩 구축 방법

  • Ollama 설치 및 설정 — Windows 10에서의 데몬 설치, Docker/WSL 대안, 권한 및 환경변수 설정 체크리스트
  • Llama 3.1 설치 및 설정 — 모델 다운로드·로컬 배포, 하드웨어 요구사항, 양자화 옵션과 호환성 가이드
  • 내부 임베딩 구축 방법 — 문서 전처리(토크나이징·청소), 배치 임베딩 생성, 임베딩 품질 검증 절차
  • 환경 결합과 최적화 전략 — GPU 활용·CPU 최적화, 양자화·프루닝, 캐싱·인덱스 튜닝 실전 팁
  • 문제 해결 및 주의사항 — 권한 오류, 메모리 부족, 버전 호환성 이슈와 빠른 대응 체크리스트
  • 실제 적용 사례 분석 — PoC 구성 예시, 성능 지표 해석, 확장 시 고려해야 할 운영 방안
